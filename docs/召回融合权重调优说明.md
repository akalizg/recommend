# 召回融合权重调优说明

## 脚本路径

```text
evaluate/tune_recall_weights.py
```

## 输入文件

```text
data/recall/als_recall.csv
data/recall/itemcf_recall.csv
data/processed/test_ratings.csv
data/features/movie_profile.csv
```

## 搜索空间

```text
als_weight: 0.0, 0.1, ..., 1.0
itemcf_weight: 1.0 - als_weight
source_bonus: 0.0, 0.05, 0.1
```

融合公式：

```text
merged_score = als_weight * normalized_als_score
             + itemcf_weight * normalized_itemcf_score
             + source_bonus * recall_source_count
```

## 选择标准

优先最大化：

```text
NDCG@10
Recall@10
Diversity@10
```

## 输出文件

```text
data/eval/recall_weight_tuning.csv
data/eval/best_recall_weights.json
data/recall/merged_recall_candidates_optimized.csv
```

不会覆盖原始：

```text
data/recall/merged_recall_candidates.csv
```
