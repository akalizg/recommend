# 推荐理由生成说明

本阶段使用本地 Qwen 模型生成推荐理由。

本地 Qwen 模型路径：

```text
D:\Javatest\pythonweb\movierec\qwen_model
```

## 运行逻辑

1. 读取现有 Top10 MMR 推荐结果，不修改推荐算法结果。
2. 合并电影标题、电影类型、用户偏好类型、评分和匹配度等已有证据。
3. 先生成模板推荐理由，保证每条推荐都有兜底解释。
4. 当 `--use-llm true` 时，优先使用本地 Qwen 将模板理由改写为更自然的中文理由。
5. 如果 Qwen 路径不存在、依赖缺失、模型加载失败、显存或内存不足、生成失败，或输出质量校验不通过，系统自动回退模板理由。

本阶段不调用 OpenAI，不调用外部在线 API，也不联网下载模型。HuggingFace Transformers 加载模型时固定使用 `local_files_only=True`。

## 默认命令

```bash
python reason/generate_reasons.py --use-llm true
```

等价于：

```bash
python reason/generate_reasons.py --use-llm true --model-path D:\Javatest\pythonweb\movierec\qwen_model
```

也可以使用封装脚本：

```bash
python scripts/run_reason_generation.py --use-llm true --model-path D:\Javatest\pythonweb\movierec\qwen_model
```

## 输出文件

```text
data/final/recommendations_with_reasons.csv
```

输出字段至少包括：

```text
userId
movieId
rank_position
rank_score
mmr_score
movie_title
movie_genres
favorite_genres
template_reason
llm_reason
final_reason
reason_source
reason_evidence
```

`reason_source` 只有两个取值：

```text
qwen
template
```

当 Qwen 生成成功且通过校验时：

```text
final_reason = llm_reason
reason_source = qwen
```

当 Qwen 不可用或生成失败时：

```text
final_reason = template_reason
reason_source = template
```
